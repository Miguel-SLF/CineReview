O Usuário avalia uma Mídia, que pode ser um Filme ou uma Série. Séries possuem Temporadas, e estas possuem Episódios. O Usuário avalia as Temporadas quando se trata de séries.

O Usuário pode ser um visitante sem cadastro, mas apenas aqueles com cadastro e logados podem avaliar obras. O Usuário tem um e-mail, senha, nome de usuário, uma lista de Avaliações atrelada a seu usuário e uma lista de mídias Favoritas. Um Usuário, também pode ser um Administrador, que pode fazer tudo que um Usuário faz, mas apenas Administradores podem cadastrar novas mídias no site.

Uma Mídia deve ter um título, gênero, sinopse, duração, classificação indicativa, data de lançamento, uma lista da Equipe (que será composta por Atores e Equipe Técnica e exibirá seus nomes completos, uma lista de funções na produção da mídia e um papel apenas se for um Ator), uma lista de Avaliações, uma nota média geral, uma nota média da narrativa, uma nota média da performance, uma nota média dos componentes visuais, uma nota média dos componentes auditivos e uma nota média da execução.

Séries são avaliadas de forma diferente, as Temporadas que são avaliadas e possuem os atributos listados para Midia, e a série ao todo recebe uma média das médias das temporadas, por exemplo, nota média de narrativa em uma série como um todo será a média da média de narrativa de todas as temporadas.

As Avaliações devem citar qual Usuário avaliou, a data da avaliação. Avaliação possui: a nota média de narrativa, que é uma média dos valores atribuídos pelo usuário para "Trama", "Ritmo", "Desenvolvimento e Diálogo de Personagens", "Construção de Mundo" e "Temática e Mensagem"; a nota média de execução, que é uma média dos valores atribuídos pelo usuário para "Atuação", "Edição" e "Direção"; a nota média de componentes visuais, que é uma média dos valores atribuídos pelo usuário para "Direção de Arte", "Cinematografia", "Cenários e Ambientes", "Figurinos e Maquiagem", "Efeitos Visuais" e "Qualidade de Imagem"; a nota média de componentes auditivos, que é uma média dos valores atribuídos pelo usuário para "Score e Soundtrack" e "Efeitos Sonoros"; e por fim, a nota média geral, que é uma média de nota média de narrativa, nota média de execução, nota média de componentes visuais e nota média de componentes auditivos.
